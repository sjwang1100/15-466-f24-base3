# Sound Pair Match
Author: Shao-Ju Wang

Design: Pair matching but using sound, distraction included.

Screen Shot:

![Screen Shot](screenshot.png)

How To Play:

Use [0-9], [a-c], depending on the level, to open cards, match the card with the same sound.
